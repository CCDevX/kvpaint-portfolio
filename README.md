A minimalist and immersive artist portfolio for **KV Paint**, a Belgian artist visual artist who blends abstract painting and macro photography.

This single-page application is built with **HTML**, **Sass**, and **JavaScript**, and uses **dynamic content injection** for smooth navigation between sections like Home, Gallery, About, and Contact.  
The site is fully **responsive** and optimized for all screen sizes.

## Features

- Minimalist & immersive design
- Color palette inspired directly by KV Paint's artwork
- SPA-like structure using JavaScript modules
- Dynamic content injection (no page reload)
- Responsive design for mobile, tablet, and desktop
- Lightbox-ready gallery layout (planned)
- Contact section with form or direct links

## Tech Stack

- HTML5
- SCSS (compiled to CSS)
- JavaScript (ES6 modules)
- Vite (optional for dev/build)
- No frameworks, just clean Vanilla JS
